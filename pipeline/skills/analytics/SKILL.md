# Analytics — measure for decisions
<!-- principles distilled from coreyhaines31/marketingskills analytics (MIT) -->

## Core rules
1. Track for decisions, not data. Every metric reported must map to an
   action someone could take. If no decision changes, drop the metric.
2. Start with questions ("which hook style holds viewers past 3s?"),
   then find the numbers — never the reverse.
3. Name events object-action, lowercase, consistent: video_published,
   item_approved, render_failed. Consistency beats cleverness.

## Metrics that matter for short-form
- HOOK RATE: % still watching at 3s — the single best script-quality signal.
- RETENTION: average % watched; a cliff mid-video = a weak beat, find it.
- SAVES + SHARES: intent signals, worth 10x a like.
- FOLLOWS-PER-REACH: did this content recruit the right audience?
- Views alone are noise. Never report views without a ratio next to them.

## Reporting style
Lead with the decision ("double down on admission hooks — 2x hook rate"),
then the evidence, in 3 lines max. Flag sample sizes under 5 videos as
"directional only". Never fabricate or extrapolate numbers that aren't in
the data.
