 HOW TO ADD THEM TO RAILWAY (1 minutes per key)
 -----------------------------------------------
 1. Open https://railway.app → your project → worker service → Variables
 2. Click "+ New Variable"
 3. Paste NAME exactly as shown (e.g. FAL_KEY) and the key as value
 4. Click Add
 5. Railway auto-redeploys after each — you can add them all then click "Redeploy" once
 6. Open https://agent-x-two.vercel.app/dashboard/models → each model should flip from
    "NO KEY" to "LIVE" or "FREE" as the worker picks them up (within 60 seconds)

 HOW TO ADD TO VERCEL (for web-side API health checks)
 -----------------------------------------------------
 Only the "AI provider wallets" balance panel needs keys on the web side too.
 1. Open Vercel → Agent-X project → Settings → Environment Variables
